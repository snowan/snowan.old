二叉树遍历(Binary Tree Traversal) - 先序, 中序, 后序, 层级遍历

与那些基本上都有标准遍历方式的线性数据结构(如链表, 一维数组)不同的是, 树结构有多种不同的遍历方式. 从二叉树的根节点出发, 节点的遍历分为三个主要步骤: 访问当前节点, 遍历当前节点左子树, 遍历当前节点右子树. 按访问节点的先后顺序, 有: 1. 深度优先 (Depth-First Order) 2. 广度优先 (Breadth-First Order)

深度优先按根节点相对于左右子节点的访问先后顺序分为: - 前序遍历(Pre-Order Traversal) - 根节点, 左子树, 右子树 (Root, Left, Right) - 中序遍历(In-Order Traversal) - 左子树, 根节点, 右子树 (Left, Root, Right) - 后序遍历(Post-Order Traversal) - 左子树, 右子树, 根节点 (Left, Right, Root)

先序遍历 (preorder):

  • 算法:

    preorder(tree)
1. 访问根节点(root node)
2. 遍历左子树, - preorder(left-subtree)
3. 遍历右子树, - preorder(right-subtree)


void preorder(TreeNode root) {
    // visit root node
    System.out.println(root.val);
    // traverse left-subtree
    preorder(root.left);
    // traverse right-subtree
    preorder(root.right);
}

Leetcode原题例子:

144. Binary Tree Preorder Traversal

Given a binary tree, return the preorder traversal of its nodes' values.

Example:

Input: [1,null,2,3]
   1
    \
     2
    /
   3

Output: [1,2,3]
Follow up: Recursive solution is trivial, could you do it iteratively?

题意: 前序遍历一个二叉树, 返回list包含所有节点值

解题: 递归(recursive)和非递归(iterative)

非递归(iterative) 用Stack (FILO), 解题步骤:

```java
// Definition for a binary tree node.
public class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode(int x) { val = x; }
}

class PreOrderTraversal {
    // recursive pre-order traversal
    public List<Integer> preorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        if (root == null) return res;
        preorder(root, res);
        return res;
    }
    private void preorder(TreeNode node, List<Integer> res) {
        if (node == null) return;
        res.add(node.val);
        preorder(node.left, res);
        preorder(node.right, res);
    } 

    // iteratively pre-order traversal
   public List<Integer> preorderTraversalIterative(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        if (root == null) return res;
        Stack<TreeNode> stack = new Stack<>();
        stack.push(root);
        while (!stack.isEmpty()) {
            TreeNode curr = stack.pop();
            res.add(curr.val);
            if (curr.right != null) stack.push(curr.right);
            if (curr.left != null) stack.push(curr.left);
        }
        return res;
    } 
}
```

中序遍历 (In-Order Traversal)

  • 算法

    inorder(tree)
1. 遍历左子树, - inorder(left-subtree)
2. 访问根节点(root node)
3. 遍历右子树, - inorder(right-subtree)


void inorder(TreeNode root) {
    // traverse left-subtree
    inorder(root.left);
    // visit root node
    System.out.println(root.val);
    // traverse right-subtree
    inorder(root.right);
}

Leetcode原题例子:

94. Binary Tree Inorder Traversal

Given a binary tree, return the inorder traversal of its nodes' values.

Example:

Input: [1,null,2,3]
   1
    \
     2
    /
   3

Output: [1,3,2]
Follow up: Recursive solution is trivial, could you do it iteratively?

题意: 中序遍历一个二叉树, 返回list包含所有节点值

解题: 递归(recursive)和非递归(iterative)

非递归(iterative) 用Stack (FILO), 解题步骤:

Java 实现:

    // recursive
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList();
        inorder(root, res);
        return res;
    }
    private void inorder(TreeNode node, List<Integer> res) {
        if (node == null) return;
        inorder(node.left, res);
        res.add(node.val);
        inorder(node.right, res);
    }

    // iterative
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        if (root == null) return res;
        Stack<TreeNode> stack = new Stack<>();
        TreeNode curr = root;
        while (curr != null || !stack.isEmpty()) {
            while (curr != null) {
                stack.push(curr);
                curr = curr.left;
            }
            curr = stack.pop();
            res.add(curr.val);
            curr = curr.right;
        }
        return res;
    }

后序遍历(Post-Order Traversal)

  • 算法

    postorder(tree)
1. 遍历左子树, - postorder(left-subtree)
2. 遍历右子树, - postorder(right-subtree)
3. 访问根节点 (root node)

void postorder(TreeNode root) {
    // traverse left-subtree
    postorder(root.left);
    // traverse right-subtree
    postorder(root.right);
    // visit root node
    System.out.println(root.val);
}

Leetcode 原题例子:

145. Binary Tree Postorder Traversal

Given a binary tree, return the postorder traversal of its nodes' values.

Example:

Input: [1,null,2,3]
   1
    \
     2
    /
   3

Output: [3,2,1]
Follow up: Recursive solution is trivial, could you do it iteratively?

题意: 后序遍历给定二叉树(root), 返回list包含所有节点

解题: 递归(recursive) 和 非递归 (iterative), 解题步骤:

Java 实现

    // recursive
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        if (root == null) return res;
        postorder(root, res);
        return res;
    }
    private void postorder(TreeNode node, List<Integer> res) {
        if (node == null) return;
        postorder(node.left, res);
        postorder(node.right, res);
        res.add(node.val);
    }

    // iterative
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        if (root == null) return res;
        Stack<TreeNode> stack = new Stack<>();
        TreeNode curr = root;
        while (curr != null || !stack.isEmpty()) {
            if (curr != null) {
                stack.push(curr);
                // root node
                res.add(curr.val);
                // right
                curr = curr.right;
            } else {
                // left
                curr = stack.pop().left;
            }
        }
        // in result order is (root, right, left), reverse ressult to pose order(left, right, root)
        Collections.reverse(res);
        return res;
    }

前序遍历, 中序遍历, 后序遍历的时间复杂度和空间复杂度分析:

时间复杂度: O(n) - n is the number of nodes

空间复杂度: O(n) - 考虑stack call functions

广度优先/层级遍历(BFS/Level Order Traversal)

  • 算法

    深度优先遍历(DFS) (递归)
1. 从第0层开始遍历, 记住每一个节点的层级(level)
2. 层级(level)相同, 就把当前节点值放入相同层
3. 遍历当前节点的下一层的左子树(level + 1)
4. 遍历当前节点的下一层的右子树(level + 1)


广度优先遍历(BFS) - 用队列(Queue)实现, 利用队列先进先出(FIFIO)特性
1. 创建队列(Queue)
2. 从根节点(root) 开始, 放入队列中
3. 遍历当前队列所有节点, 所有节点在相同层
4. 把相同层的当前节点的下一层即左右子树放入队列

Leetcode 原题例子分析.

102. Binary Tree Level Order Traversal

Given a binary tree, return the level order traversal of its nodes' values. (ie, from left to right, level by level).

For example:
Given binary tree [3,9,20,null,null,15,7],

    3
   / \
  9  20
    /  \
   15   7
return its level order traversal as:

[
  [3],
  [9,20],
  [15,7]
]

题目: 层级遍历一颗给定的二叉树, 一层一层打印出二叉树的所有节点

解题: DFS and BFS

Java 实现

    // DFS (recursive)
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        if (root == null) return res;
        helper(root, res, 0);
        return res;
    }
    private void helper(TreeNode root, List<List<Integer>> res, int level) {
        if (root == null) return;
        if (res.size() <= level) {
            res.add(new ArrayList<>());
        }
        res.get(level).add(root.val);

        helper(root.left, res, level + 1);
        helper(root.right, res, level + 1);
    }

    // BFS, using queue
    public List<List<Integer>> levelOrderBFS(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        if (root == null) return res;
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        while (!queue.isEmpty()) {
            int size = queue.size();
            List<Integer> level = new ArrayList<>();
            while (size-- > 0) {
                TreeNode curr = queue.poll();
                level.add(curr.val);
                if (curr.left != null) queue.offer(curr.left);
                if (curr.right != null) queue.offer(curr.right);
            }
            res.add(level);
        }
        return res;
    }

广度优先遍历(BFS)/层级遍历(Level Order Traversal) 复杂度分析:

时间复杂度: O(N) - N 节点的个数, 每个节点遍历一次

空间复杂度: O(N)
