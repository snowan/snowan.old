Leetcode 792 Number of Matching Subsequences 的解题报告

Posted by     "Snowan" on Tuesday, June 4, 2019

Question

Given string S and a dictionary of words words, find the number of words[i] that is a subsequence of S.

Example :
Input: 
S = "abcde"
words = ["a", "bb", "acd", "ace"]
Output: 3
Explanation: There are three words in words that are a subsequence of S: "a", "acd", "ace".

Note:

  • All words in words and S will only consists of lowercase letters.
  • The length of S will be in the range of [1, 50000].
  • The length of words will be in the range of [1, 5000].
  • The length of words[i] will be in the range of [1, 50].

题的意思是: 给一个很长的字符串S, 和一个字符串数组words, 求字符串数组中字符是S的连续子序列, (下面是维基百科的定义)

在数学中，某个序列的子序列是从最初序列通过去除某些元素但不破坏余下元素的相对位置（在前或在后）而形成的新序列。

正式地说，假设 X 是集合而 (ak)k ∈ K 是 X 中的序列，其中若 (ak) 是有限序列，则 K = {1,2,3,…,n}；若 (ak) 是无限序列，则K = \mathbb{N} 。则 (ak) 的子序列是形如 {\displaystyle (a{n{r)))} 的序列，这里的 (nr) 是在索引集合 K 中严格递增序列

解答

解法1:

首先能想到的是暴力解法, 对于每个words中字符, 判断其是否是S的subsequence, 如果是res就加1.

Pros: 解法简单, 容易想到

Cons: 这样时间复杂度很高, 而且按照题中Note中, S的长度要远大于words中字符的长度

时间复杂度 (TC):

O(m * n) - m is words length, n is S length

空间复杂度 (SC):

O(1)

以下是Java 版本的实现:

public int numMatchingSubseq(String S, String[] words) {
        if (S == null || S.length() == 0 || words == null || words.length == 0) return 0;
        int res = 0;
        for (String w : words) {
            if (isSubseq(S, w)) res++;
        }
        return res;
    }
    private boolean isSubseq(String s, String t) {
       if (t.length() > s.length()) return false;
        int n = s.length();
        int m = t.length();
        int i = 0;
        int j = 0;
        int count = 0;
        while (i < n && j < m) {
          // rest cannot match, early termination
          if (n - i < m - j) return false;
          if (s.charAt(i++) == t.charAt(j++)) {
            count++;
          } else {
            i++;
          }
        }
        return count == m;
    }

解法2

我们知道S的长度要远大于words的长度, 所以我们应该尽量只遍历一遍S, 这样的话, 我们就可以提高performance. 这里参考大神 StefanPochmann 讨论里提出的解法.

思路如下:

按words中每个word的第一个字符分组,

遍历S中每一个字符ch,然后匹配以ch开头的组, 删除当前匹配的字符, 生成新的word保存到新的以当前word第一个字符分组中. 如果删除第一个字符后, 新的word长度为0, 那么word是S的subsequence, 结果加1.

以题中给的例子详细解答过程:

S = "abcde"
words = ["a", "bb", "acd", "ace"]

1. 按words中每个word第一个字符分组:
  ['a', {"a", "acd", "ace"}]
  ['b', {"bb"}]
2. 遍历S
  - 当前字符: ‘a', 拿到分组中’a‘组strings,{"a", "acd", "ace"} 
    - 把‘a’分组中第一个字符删除, 组成新的分组, 判断新的string长度是否为0, “a”删除’a‘为0, res++. 
    - 得到新的分组: res = 1
      ['c', {"cd", "ce"}]
      ['b', {"bb"}]
  - 当前字符: ‘b', 拿到分组’b'中strings, {"bb"}
    - 删除‘b', 重新分组
    - 得到新的分组: res = 1
      ['c', {"cd", "ce"}]
      ['b', {"b"}] 
  - 当前字符: ‘c’
    - 删除‘c', 重新分组
    - 得到新的分组: res = 1
      ['d', {"d"}]
      ['b', {"b"}]  
      ['e', {"e"}]
  - 当前字符: ‘d’
    - 删除‘d', 重新分组
    - 得到新的分组: res = 2
      ['b', {"b"}]  
      ['e', {"e"}]
  - 当前字符: ‘e’
    - 删除‘e', 重新分组
    - 得到新的分组: res = 3
      ['b', {"b"}]'
遍历S结束, 得到结果, 剩下的分组都不是S的subsequence.

Java 解法:

public int numMatchingSubseq(String S, String[] words) {
    if (S == null || S.length() == 0 || words == null || words.length == 0) return 0;
    int res = 0;
    Map<Character, List<String>> map = new HashMap<>();
    for (String w : words) {
      char ch = w.charAt(0);
      List<String> list = map.getOrDefault(ch, new ArrayList<>());
      list.add(w);
      map.put(ch, list);
    }
    for (char ch : S.toCharArray()) {
      if (!map.containsKey(ch)) continue;
      List<String> curr = map.get(ch);
      map.remove(ch);
      for (String s : curr) {
        String tmp = s.substring(1);
        if (tmp.length() == 0) {
          res++;
        } else {
          List<String> next = map.getOrDefault(tmp.charAt(0), new ArrayList<>());
          next.add(tmp);
          map.put(tmp.charAt(0), next);
        }
      }
    }
    return res;
  }

Python 解法 (谢谢 sunboman Contribute. :-) )

import collections

class Solution:
    def numMatchingSubseq(self, S: str, words: List[str]) -> int:
        word_dict = collections.defaultdict(list)
        for word in words:
            word_dict[word[0]].append(word[1:])
        for c in S:
            for word in word_dict.pop(c, []):
                if not word:
                    word_dict[''].append('')
                else:
                    word_dict[word[0]].append(word[1:])
        return len(word_dict[''])
